Foodie Delight - Project Details

About the Project

Foodie Delight is an online food ordering platform that allows users to browse restaurant menus, place orders, and make secure payments. It provides a seamless and user-friendly experience for food lovers.

Functionality

  • Browse restaurant menus and select food items
  • Secure authentication and user profile management
  • Cart and order management with real-time updates
  • Online payments using multiple payment gateways

Technologies Used

Django Rest Framework PostgreSQL Tailwind CSS Bootstrap JavaScript
