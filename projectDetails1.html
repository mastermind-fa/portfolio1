YouFashion - Project Details

About the Project

YouFashion is an online fashion store designed to provide users with a seamless shopping experience. It offers trendy clothing items, a secure authentication system, cart management, and an easy checkout process with multiple payment options.

Functionality

  • Browse and purchase trendy fashion items
  • Secure authentication and user management
  • Cart and order management
  • Seamless checkout with multiple payment options

Technologies Used

Django Rest Framework PostgreSQL Tailwind CSS Bootstrap JavaScript
